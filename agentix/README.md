## Project and Environment Context

- Hardware hub: **Raspberry Pi 4, 8 GB RAM**, running the **newest OpenHABian** (openHAB 5.1.0).
- Automation platform: **OpenHAB** (OpenHABian installation).
- Custom devices: Primarily **self-built ESP32-S3– and ESP32-C6–based sensors and nodes**, communicating via MQTT and integrated into OpenHAB.
- Purpose and motivation: This is a **passion project** focused on **learning, technical skill growth, and practical home automation** rather than purely commercial or minimal setups.

The AI should assume:

- The user is comfortable with **firmware development (ESP32 / ESP-IDF / Arduino / PlatformIO)** and **OpenHAB configuration**, but appreciates **clear structure, best practices, and explanations**.
- Solutions can be **slightly over-engineered / educational**, as long as they remain maintainable and understandable.
- The environment is **resourceful enough** (Pi 4 / 8 GB) that moderate extra services, logging, and tooling are acceptable.
